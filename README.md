# MBac-Quick-Access
It makes easier to access dropbox pages in your ManageBac.

## Installation

1. Go to the `Releases` section in this repository.
2. Download the ZIP file.
3. Unzip it.
4. Go to the Extensions page in your browser (Google Chrome is recommended).
5. Load the unzipped folder. (In Google Chrome, turn on Developer Mode using the toggle in the top-right corner, then click “Load unpacked.”)
6. Try it.

## How to Use

When you click the `Submit Coursework button`, press the `X` key on your keyboard at the same time. Then you can directly access the Dropbox page for that task.
