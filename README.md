# MBac-Quick-Access
It makes easier to access dropbox pages in your ManageBac.

## Installation
1. Go to `releases` in this repository
2. Download a zip file.
3. Unzip it.
4. Go to the extension page in your browser (recommend Google Chrome).
5. Load the unzipped folder. (In case of Google Chrome, turn on developer mode from the top-right toggle, and load unpacked folders.)
6. Try it.

## How to Use
When you click bottom `Submit Coursework`, press `x` key in your keyboard at the same time.
Then you can directly access a dropbox page of that task.
